## Girit Exam

---

### Mandatory Requirements

- PHP >= 8.2.x
- Node >= 19.4.x
- Composer >= 2.5.x

### Installation

1. Change base url in `.env` to your base app url:
```dotenv
APP_URL=http://girit.loc/
```

2. Run following commands:
```text
npm install
composer install
```

### Running

Open new terminal and run
```text
npm run dev
```

Open new terminal and run
```text
php artisan serve
```
